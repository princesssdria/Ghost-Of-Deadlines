# Ghost-Of-Deadlines

A spooky, web-based assignment tracker titled **Ghost of My Deadlines**.

## Features
- Assignment input form with Task Name, Due Date, and Priority.
- Dynamic Ghost tier system that escalates with nearest effective deadline (`High` priority applies a `-2 day` penalty).
- Doom Clock header countdown to the closest assignment.
- Curse Meter that reflects ratio of high-priority tasks.
- Canvas fog particle overlay that scales with active assignment count.
- Ghost Chat bubble with rotating snarky messages.
- Graveyard footer where exorcised tasks become tombstones with epitaph tooltips.
- Ambient horror soundscape (wind loop + creak effect on exorcise) with click-gated autoplay compliance and mute skull toggle.
- LocalStorage persistence for active tasks, graveyard entries, and mute state.

## Run
Open `index.html` in a browser.
