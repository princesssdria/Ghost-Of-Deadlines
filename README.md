# Ghost-Of-Deadlines

A web-based task management dashboard titled **Ghost of My Deadlines**.

## Features
- Add assignments with task name, due date, and priority.
- Dynamic Ghost Threat UI that evolves across four tiers based on nearest deadline.
- High priority assignments accelerate threat by reducing effective days by 2.
- Exorcise button removes completed tasks with a fade animation.
- Data persistence using LocalStorage for assignments and ghost tier state.

## Run
Open `index.html` in a browser.
